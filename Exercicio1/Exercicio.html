Meta tag Viewpoint



Os navegadores de dispositivos móveis tentam exibir páginas web feitas somente para desktop (a maioria delas), ajustando automaticamente, o zoom do display, e isso pode ser problemático para os sites que já foram planejados para telas pequenas. Para resolver isso surgiu a meta tag viewport.

Sua sintaxe é: meta name="viewport" content=""

No atributo content, é possível declarar uma diversidade de parâmetros e valores conforme o tipo de visualização que se configurar às paginas. Ou seja, com ela é possível mostrar “resoluções personalizadas” aos visitantes para determinados dispositivos."
